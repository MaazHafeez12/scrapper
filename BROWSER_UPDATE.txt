ADVANCED EXAMPLE:
-----------------
import asyncio
from scrapers.playwright_scraper import PlaywrightScraper

async def custom_scrape():
    scraper = PlaywrightScraper()
    await scraper.setup_browser()
    
    # Navigate
    await scraper.goto_page('https://indeed.com')
    
    # Type text
    await scraper.type_text('input#search', 'engineer')
    
    # Press Enter
    await scraper.press_key('Enter')
    
    # Wait for results
    await scraper.wait_for_selector('.job-card')
    
    # Scroll
    await scraper.scroll_page(scrolls=3)
    
    # Screenshot
    await scraper.screenshot('results.png')
    
    # Get HTML
    html = await scraper.get_html()
    
    await scraper.close_browser()

asyncio.run(custom_scrape())

TROUBLESHOOTING:
================

Selenium Issues:
----------------
Q: ChromeDriver not found?
A: Auto-installed via webdriver-manager

A: Use wait_for_element():
   scraper.wait_for_element(By.CLASS_NAME, 'job-card')

------------------
Q: Browser not installed?
A: Run: python -m playwright install chromium

Q: Async errors?
A: Use the sync wrapper:
   jobs = scraper.scrape_jobs(filters)  # Not async!
